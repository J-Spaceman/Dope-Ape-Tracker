# Dope Ape Club — Wallet Intelligence Tracker

Real-time NFT ownership cluster graph for the Dope Ape Club collection.

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to vercel.com → Import Project → select the repo
3. Vercel auto-detects Vite — click Deploy
4. Done. Your URL is live.

## Local Development

```bash
npm install
npm run dev
```
